git命令： 1、git仓库初始化： git init 清除终端窗口内容：cls 查看文件状态： git status -s 2、将未跟踪、修改后的文件添加到暂存区： git add 文件名称（单个文件）、git add .全部文件) 3、将暂存区的文件添加到git仓库： git commit -m '提交消息' 4、将工作区修改的文件直接添加到仓库（未跟踪文件不可使用）： git commit -a -m '提交消息' 5、查看提交记录： git log 查看所有的提交记录 git log -2 查看最近的两条提交记录 git reflog 查看所有提交记录(简写) 6、根据提交状态ID回滚代码： git reset --hard 提交命令ID ================================================================================ 1、将暂存区的文件移除： git reset HEAD 文件名称（单个文件）/.(全部文件) 2、将git仓库文件覆盖工作区文件（不可恢复，慎用）： git checkout -- 文件名 3、删除git仓库文件： git rm -f 文件名称 ------------- 从 Git 仓库和工作区中同时移除对应的文件 git rm --cached 文件名称 ------------- 只从 Git 仓库中移除指定的文件，但保留工作区中对应的文件 ================================================================================
