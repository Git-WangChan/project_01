git命令： 1、git仓库初始化： git init 清除终端窗口内容：cls 查看文件状态： git status -s 2、将未跟踪、修改后的文件添加到暂存区： git add 文件名称（单个文件）、git add .全部文件) 3、将暂存区的文件添加到git仓库： git commit -m '提交消息' 4、将工作区修改的文件直接添加到仓库（未跟踪文件不可使用）： git commit -a -m '提交消息' 5、查看提交记录： git log 查看所有的提交记录 git log -2 查看最近的两条提交记录 git reflog 查看所有提交记录(简写) 6、根据提交状态ID回滚代码： git reset --hard 提交命令ID ================================================================================ 1、将暂存区的文件移除： git reset HEAD 文件名称（单个文件）/.(全部文件) 2、将git仓库文件覆盖工作区文件（不可恢复，慎用）： git checkout -- 文件名 3、删除git仓库文件： git rm -f 文件名称 ------------- 从 Git 仓库和工作区中同时移除对应的文件 git rm --cached 文件名称 ------------- 只从 Git 仓库中移除指定的文件，但保留工作区中对应的文件 ================================================================================ ===================== 两种方法将本地仓库关联到Github ================================= git remote add origin https://github.com/Git-WangChan/project_01.git git push -u origin master --------------------------------------------------------------------------------- 1. https 1.1 创建项目文件夹 1.2 初始化git本地仓库，并确保工作目录是干净的状态 1.3 创建github仓库 1.4 将本地仓库关联到github仓库上,并设置远程仓库的别名 git remote add origin https://github.com/leelon/project_01.git 1.5 将本地仓库的代码 第一次推送到远程仓库中 git push -u origin master 1.6 当修改本地文件后，需要再次提交到本地仓库 1.7 将本地仓库修改的文件提交到远程仓库 git push ------------------------------------------------------------------------------ 2.
